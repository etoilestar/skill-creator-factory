# 当前时间查询 Skill

## 输入要求

用户输入“当前时间”。

## 工作流程

1. 解析用户的输入请求。
2. 执行查询当前时间的脚本。
3. 返回结果。

## 能力调度说明

当前调度模型负责理解用户需求并调用相应的脚本来查询时间。

当需要获取当前系统时间时，必须运行真实存在的脚本：

```bash
python scripts/get_current_time.py
```

## 资源使用说明

执行时间查询的代码在 `scripts/` 目录中的脚本文件中。

## 输出要求

输出格式为字符串，形式为“YYYY-MM-DD HH:MM:SS”。

## 依赖与配置

无。

## 失败处理

如未能获取时间，则返回错误信息提示用户。
