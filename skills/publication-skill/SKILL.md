# 公文写作 Skill

## 使用场景

当用户需要撰写公文概要时被触发，例如用户提供公文的要点或主题。

## 工作流程

1. 读取用户输入。
2. 按规则处理输入生成公文概要。
3. 输出用户要求的结果到终端。

## 依赖

仅使用 Python 标准库。

## 使用方式

```bash
python scripts/main.py
```

## 输出要求

- 使用中文输出。
- 报错信息必须使用中文。
- 不输出无关解释。
