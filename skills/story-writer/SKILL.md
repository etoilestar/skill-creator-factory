# 写故事 Skill

## 使用场景

此 Skill 适用于编写儿童故事，特别是关于主题为勇气的小狐狸的故事。

## 不适用场景

当用户希望生成非故事类文本或没有指定主题时，不应使用此 Skill。

## 输入要求

用户需要提供故事的主题，例如“勇气的小狐狸”。

## 工作流程

1. 用户输入主题。
2. Skill 根据主题生成故事内容。
3. 将故事内容格式化为 Word 文档，包含以下格式要求：
   - 大标题3号字体
   - 小标题4号字体
   - 正文小四，字体为宋体，单倍行距
4. 输出生成的 Word 文档。

## 资源使用说明

- 此 Skill 主要依赖于内部逻辑生成故事，不会调用外部资源。
- 格式要求将创作在生成的 Word 文档中实现。

## 输出要求

输出为 Word 文档，包含故事内容，格式应符合以下要求：
- 大标题3号字体
- 小标题4号字体
- 正文小四，字体为宋体，单倍行距

## 失败处理

如果主题输入不明确，系统会提示用户重新输入主题。
